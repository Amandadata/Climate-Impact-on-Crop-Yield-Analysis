# Climate-Impact-on-Crop-Yield-Analysis
Climate impact on crop yield project using Excel 
🌾 Crop Yield Analysis and Optimization Strategies
📌 Project Summary
Analyzed agricultural data to identify how environmental factors such as temperature, rainfall, CO₂ emissions, and pesticide usage impact crop yield. Delivered actionable insights to optimize productivity and promote sustainable farming practices.
🎯 Business Problem
Farmers face inconsistent crop yields due to changing climate conditions and resource mismanagement.
This project aims to identify key factors affecting yield and provide data-driven solutions to improve agricultural output.
🔍 Approach & Steps
Data Collection – Used dataset containing crop yield, climate, and farming variables
Data Cleaning – Handled inconsistencies and prepared structured data in Excel
Data Analysis –Used Pivot Tables to identify trends
Compared yield across temperature, rainfall, and farming practices
Data Visualization – Created charts to highlight patterns and relationships
Insight Generation – Identified optimal ranges and negative impacts
Recommendations – Suggested strategies to improve yield and sustainability
📊 Key Insights
Crop yield peaks at 25°C and declines beyond this temperature
Optimal rainfall range is 2200–2700 mm
Increased CO₂ emissions negatively impact yield
Excess pesticide use reduces soil health
Drought-resistant crops show better performance under stress
💡 Solutions & Recommendations
Adopt climate-resilient crop varieties
Implement efficient irrigation systems (drip/sprinkler)
Reduce pesticide usage using sustainable practices (IPM)
Improve soil health with organic inputs and crop rotation
